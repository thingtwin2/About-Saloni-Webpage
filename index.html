Find the blob

This assignment was done in week two of GWC.For this project, I worked with Molly M. It was really fun to do this assignment, because once we found the mistake in our code, it worked perfectly. The way this works is, at first the user enters the color they would like to find, and then the scribbler robots turns and finds that color, by taking pictures with the camera on the front of it, and when it identifies the color, it moves toward it. This project was coded in Python. Originally this project was done with the emulator, an online version of the scribbler robots. On the emulator the code worked perfectly, and every time the scribbler robot found the right color blob. Then we tried using the sensors on the real life scribbler robots, to mimic the same process. But the camera is not very good at identifying the correct colors. So instead of finding a certain color, the robot find an object within a certain distance. We called this "Find the Box".
